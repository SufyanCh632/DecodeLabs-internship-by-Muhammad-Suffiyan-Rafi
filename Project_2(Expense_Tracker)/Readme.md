# Expense Tracker Flask App

A simple Expense Tracker web application built using Python and Flask.
Users can add expenses, view all entered expenses, and automatically calculate the total amount spent.

---

# Features

* Add expense amounts
* View expense history
* Automatic total calculation
* Simple and clean web interface
* Beginner-friendly Flask project

---

# Technologies Used

* Python
* Flask
* HTML
* CSS

# Installation Guide

## 1. Clone or Download the Project

```bash
git clone <your-repository-link>
```

Or simply download the ZIP file.

---

## 2. Install Flask

Make sure Python is installed on your system.

Install Flask using pip:

```bash
pip install flask
```

---

# Running the Application

Run the Flask application using:

```bash
python app.py
```

After running the command, open your browser and visit:

```text
http://127.0.0.1:5000
```

---

# How It Works

1. User enters an expense amount.
2. The expense is added to the list.
3. The application calculates the total spending.
4. All expenses are displayed on the webpage.

---
