# To-Do-Project
# 📝 Python To-Do List (Console Application)

A beginner-friendly **To-Do List application** built using Python.
This project focuses on core programming concepts like lists, loops, and user input handling.

---

## 🚀 Features

* ✅ Add tasks
* 📋 View all tasks
* ⚡ Simple and fast console-based interface
* 🧠 Easy to understand logic for beginners

---

## 🧠 Concepts Covered

* Python Lists (`append`)
* Loops (`for`, `while`)
* Conditional Statements (`if-else`)
* Functions (optional structure)
* IPO Model:

  * Input → User enters task
  * Process → Store task in list
  * Output → Display tasks
